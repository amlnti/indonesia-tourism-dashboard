# 🇮🇩 Indonesia Tourism Dashboard

An interactive Business Intelligence dashboard built with **Microsoft Power BI** to analyze tourism destinations in Indonesia. This project visualizes tourism data from five major cities using interactive charts, KPI cards, maps, tables, and slicers to generate meaningful insights.

---

## 📌 Project Overview

The purpose of this project is to transform raw tourism data into an interactive dashboard that helps users analyze tourism destinations based on city, category, and visitor information. The dashboard enables users to identify tourism patterns and compare destinations through visual analytics.

## 📊 Dataset

**Dataset Name:** Indonesia Tourism Destination

**Source:**
https://www.kaggle.com/datasets/aprabowo/indonesia-tourism-destination

The dataset contains information about tourism destinations in five major cities in Indonesia:

- Jakarta
- Bandung
- Semarang
- Yogyakarta
- Surabaya

### Dataset Files

| File | Description |
|------|-------------|
| tourism_with_id.csv | Information about tourism destinations, including city, category, location, and other destination details. |
| tourism_rating.csv | User ratings for each tourism destination, used for recommendation analysis. |
| user.csv | Dummy user data for recommendation system development. |
| package_tourism.csv | Tourism package recommendations based on travel time, cost, and rating. |

---

## 🔄 ETL Process

The dataset was processed using Power Query before visualization.

The ETL process includes:

- Importing CSV files into Power BI
- Cleaning unnecessary data
- Changing data types
- Preparing latitude and longitude for map visualization
- Creating relationships between tables
- Loading transformed data into the Power BI model

## 📈 Dashboard Features

The dashboard includes several interactive visualizations:

### KPI Cards
- Total Tourism Destinations
- Total Visitors
- Tourism Destinations by City
- Tourism Destinations by Category

### Charts
- Tourism Destinations by Category
- Tourism Destinations by City
- Top 3 Most Visited Destinations

### Map Visualization
- Geographic distribution of tourism destinations across Indonesia.

### Table
- Destination details including city, category, and rating.

### Interactive Filters
- Filter by City
- Filter by Tourism Category

---

## 💡 Insights

Some insights obtained from the dashboard include:

- Yogyakarta has the highest number of tourism destinations among the five cities.
- Taman Hiburan is the category with the largest number of destinations.
- Users can easily compare tourism data between cities using interactive filters.
- Map visualization helps identify the geographical distribution of tourism destinations.
